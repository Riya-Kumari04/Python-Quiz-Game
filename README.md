# 🐍 Python Quiz Game

A console-based Python Quiz Game developed using **Python**, **Object-Oriented Programming (OOP)**, **JSON**, and **File Handling**. The application allows users to test their Python knowledge through multiple quiz categories while tracking scores, accuracy, and performance statistics.

---

## 📌 Features

- 👤 Player Name Registration
- 📚 Multiple Quiz Categories
  - Basics
  - Data Structures
  - Object-Oriented Programming (OOP)
  - Advanced Python
- 🎯 Difficulty Levels (Easy, Medium, Hard)
- 🔀 Randomized Questions
- 📊 Progress Bar
- ✅ Instant Answer Validation
- 💡 Explanation After Every Question
- 🔥 Answer Streak Tracking
- 📈 Accuracy Calculation
- 🏆 High Score Management
- 📝 Answer Review at the End
- 🎨 Colored Console Interface
- 💾 JSON-based Question and Score Storage
- 🏗️ Modular Project Structure
